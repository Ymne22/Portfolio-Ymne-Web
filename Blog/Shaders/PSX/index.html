Overview

The PSX Shader Pack is a collection of stylized shaders designed to replicate the specific technical artifacts of the original PlayStation. Originally developed for Unity Built-in Render Pipeline (BRP), the project has since been ported to Godot 4.5, offering a consistent retro aesthetic across both engines.

PSX Style Preview — Vertex jittering and color quantization in action

Vertex Jitter & Affine Distortion

One of the most defining characteristics of the PS1 era is the "wobbly" geometry. This is caused by the console's lack of sub-pixel precision and float conversion.

In these shaders, we simulate this by snapping vertex positions in screen space. Instead of using post-processing, the jitter is calculated per-material, allowing for selective application and finer control over intensity. In both Unity and Godot implementations, the vertex shader performs a perspective-correct division and rounding to mimic the hardware's integer-based coordinate system.

Color Quantization & Dithering

To replicate the limited bit-depth of the PSX's frame buffer, we implement Color Quantization. This reduces the available color palette, leading to visible banding.

To soften these bands, we use Bayer Matrix Dithering. This technique inserts a checkerboard-like pattern of pixels to create the illusion of more colors, a common trick used in the 90s.

Post-Processing Stack

The pack includes a dedicated post-FX sequence that handles the final screen look:

  • Resolution Downscaling: Mimics the low native resolution (typically 320x240 or 640x480).
  • Depth-Based Fog: A custom fog implementation that works with the camera's depth texture, including skybox override support to prevent the fog from obscuring the distant horizon.
  • Bayer Filter: A global dithering pass applied to the entire screen.

Specialized Materials

Beyond standard opaque surfaces, the pack includes specialized shaders like the PSX Water Shader. This material features:

  • UV Distortion: Animated surface waves driven by sine functions.
  • Color Banding: Internal quantization for that classic "flat" look.
  • Vertex Offset ripples: Low-poly wave movements that sync with the jitter effect.
