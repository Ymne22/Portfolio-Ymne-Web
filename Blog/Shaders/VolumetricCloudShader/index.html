Overview

The Volumetric Cloud Shadow shader provides a performant way to render 3D-feeling cloud shadows within a defined volume. Instead of standard 2D projections, it uses a cube mesh to define a bounding region where shadows are calculated based on depth-reconstructed world positions.

This technique is ideal for diorama-style games or localized environmental effects where you need a "volume" of darkening without the overhead of true volumetric raymarching.

Depth-Based Reconstruction

The core of the effect relies on knowing where in the world the background pixels are. We sample the _CameraDepthTexture and use the camera's view matrix to project that depth back into world space.

HLSL 
// Reconstruct World Position
float rawDepth = SAMPLE_DEPTH_TEXTURE(_CameraDepthTexture, uv);
float linearDepth = LinearEyeDepth(rawDepth);

float3 cameraForward = -UNITY_MATRIX_V[2].xyz;
float viewZ = dot(rayDir, cameraForward);
float dist = linearDepth / viewZ;

float3 worldPos = _WorldSpaceCameraPos + rayDir * dist;

Object-Space Projection

By converting the reconstructed world position into the Object Space of the cube mesh, we can easily check if a pixel is "inside" the volume. The cube's unit bounds (-0.5 to 0.5) serve as our coordinate system for noise sampling.

HLSL 
// Object Space Projection
float3 objPos = mul(unity_WorldToObject, float4(worldPos, 1.0)).xyz;

// Bounds Check
float3 distToEdge = 0.5 - abs(objPos);
float minEdgeDist = min(distToEdge.x, min(distToEdge.y, distToEdge.z));
if (minEdgeDist < 0) return fixed4(1, 1, 1, 1); // Outside volume

Procedural 3D Noise

The cloud pattern is generated using a simple 3D value noise function. We sample this noise using the object-space coordinates, offset by time to simulate wind.

HLSL 
float3 noisePos = (objPos * _NoiseScale.xyz) + (_Speed.xyz * _Time.y);
float n = noise(noisePos);

// Contrast Adjustment
n = saturate((n - 0.5) * _Contrast + 0.5);

Soft Edge Masking

To avoid harsh clipping at the edges of the cube, we compute a smoothstep mask based on the distance to the nearest cube face.

HLSL 
float edgeMask = smoothstep(0, _EdgeSoftness, minEdgeDist);
float shadowFactor = n * _Intensity * edgeMask;
fixed3 finalColor = lerp(fixed3(1,1,1), _ShadowColor.rgb, saturate(shadowFactor));
