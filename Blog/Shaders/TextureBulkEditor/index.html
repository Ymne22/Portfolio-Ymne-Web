Overview

Managing texture import settings in Unity across dozens or hundreds of assets can be a tedious and error-prone process. The Texture Bulk Editor is a custom Unity Editor tool designed to streamline this workflow, allowing developers to batch-modify import settings with precision and efficiency. For using this tool, simply download the script and select the texturebulkeditor on tool dropdown section.

Key Features

  • Batch Processing: Modify multiple textures simultaneously, saving hours of manual work.
  • Flexible Selection: Add textures via project selection, folder scanning, or simple drag-and-drop.
  • Selective Overrides: Each setting (Texture Type, Max Size, Compression, etc.) can be toggled individually.
  • Platform Specificity: Full support for per-platform overrides (Standalone, Android, iOS, Switch, etc.).
  • Undo Support: Integrated with Unity's Undo system for safe experimentation.
  • Performance Mode: Disable thumbnails to maintain editor responsiveness when handling thousands of assets.

The Workflow

The tool is designed to fit naturally into any production pipeline:

  1. Load: Use the "Add Selected" or "Add Folder" buttons to populate the list. You can also drag files directly into the drop zone.
  2. Filter: Uncheck items you don't want to modify, or use the "Remove Unchecked" button to focus only on specific assets.
  3. Configure: Enable the override toggles for the settings you wish to change. Grayed-out settings will remain untouched.
  4. Apply: Review the summary and hit Apply. The tool reimports assets in a single batch edit operation for maximum speed.

Platform Overrides

One of the tool's most powerful capabilities is managing platform-specific settings. You can define unique Max Size, Texture Format, and Compression Quality for every target platform supported by your project.
