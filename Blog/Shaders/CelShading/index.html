Overview

The Cel Shading Shader is a custom-built solution for Unity's Built-in Render Pipeline, specifically designed to achieve the "flat" but depth-rich look of modern anime and MMD-style character models.

While standard Unity shaders focus on realistic PBR gradients, this shader uses stepped lighting calculations to create hard-edge shadows, while still allowing for subtle light bleeds and stylized rim highlights that define the character silhouette.

Shading Model

The core of the shader is a multi-step shadow threshold. By evaluating the dot product between the surface normal and the light direction, we can "snap" the resulting value into discrete levels.

  • Hard-Edge Shadows: Uses a sharp step function to create clean transitions between light and dark areas.
  • Shadow Steps: Adjustable granularity allows for 2-tone (classic anime) or multi-tone (modern cinematic) cel shading.
  • Specular Control: Highlights are treated with the same stepped logic, creating sharp, non-blurry shines on metallic or glossy surfaces.

Subsurface Scattering (SSS)

To prevent skin and thin materials from appearing completely flat and lifeless, the shader includes a simulated Subsurface Scattering pass.

This approximates light bleeding through edges and thin geometry (like ears or folded fabric) by adding a customizable color bleed based on the light's inverse direction. It creates a warm, "soft" feel to the character's skin without requiring expensive raymarching or high-poly buffers.

SSS and Rim Light Detail — Enhancing depth and silhouette clarity

Rim Lighting

The rim light implementation is highly stylized. It can be configured to only appear in the shadowed areas of the character, which helps maintain the silhouette even when the main light source is positioned behind the camera. This is a common technique used in professional anime productions to give characters a 3D feel while keeping them visually integrated with the 2D background.
