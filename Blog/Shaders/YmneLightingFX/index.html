Overview

Ymne LightingFX is a comprehensive collection of advanced rendering effects and shaders designed specifically for Unity's Built-in Render Pipeline (BRP). The suite brings modern graphical features to the legacy pipeline, focusing on performance, visual fidelity, and flexibility.

The project includes a full post-processing stack, volumetric atmospheric systems, and specialized shaders that extend the capabilities of BRP to match modern industry standards.

Important Notes

Graphics API: These rendering systems are optimized and tested with Vulkan. Some features (especially voxel lighting and temporal reprojection) may behave incorrectly or fail under DirectX 11/12. It is highly recommended to switch your Graphics API to Vulkan in Project Settings.

Lighting Stack

The lighting stack provides several screen-space effects that dramatically enhance the depth and realism of the scene. All effects are integrated with a shared temporal reprojection buffer for maximum stability.

SSGI and SSAO Preview — Real-time diffuse bounce and ambient occlusion

1. Screen-Space Global Illumination (SSGI)

Simulates diffuse bounce lighting from surfaces in the view. It features adjustable sample counts, ray distance, and thickness, stabilized by temporal reprojection and smoothed with Gaussian filtering.

2. Screen-Space Ambient Occlusion (SSAO)

A high-quality hemisphere sampling implementation with controllable radius and power. It uses depth and normal-weighted filtering to maintain sharp edges while reducing noise.

3. Screen-Space Reflections (SSR)

Raymarched reflections that react to surface roughness. It includes contrast controls for glossy versus rough surfaces and denoising via temporal blending and Gaussian blur.

SSR Preview — Roughness-based raymarched reflections

4. Temporal Reprojection

A core component of the stack that provides stability across multiple frames. It features an adjustable blend factor and automatic history resets on camera jumps to prevent ghosting.

Volumetric Atmosphere

This system handles the global environment, atmospheric scattering, and large-scale volumetric effects.

  • Volumetric Clouds: Raymarched layers with coverage, density, and detail noise. Supports wind animation, silver lining effects, and HG scattering.
  • Global Volumetric Fog: Height-based exponential fog with independent day/night settings and skybox integration.
  • God Rays: Volumetric light shafts from the sun with adjustable intensity, decay, and temporal smoothing.

Volumetric Clouds — Dramatic atmospheric scattering and silver linings

Voxel Volumetric Light

Unlike screen-space god rays, this system provides 3D volumetric scattering for local point and spot lights using a voxelized grid.

It supports both Baked mode (precomputed 3D textures) and Realtime updates via GPU compute shaders. Visuals can be fine-tuned with density, extinction, anisotropy, and 3D noise modulation for turbulent light shafts.

Voxel Volumetric Light — Per-light scattering with 3D noise modulation

Shadow CSM Settings

A specialized shader utility for Unity's Cascade Shadow Maps (CSM). It implements soft shadows via rotated Poisson disk sampling and provides smooth cascade blending to eliminate hard transitions between shadow levels.

PBR GGX Deferred Shader

A deferred-capable version of the suite's custom PBR shaders. It implements the GGX BRDF with a metallic/roughness workflow, supporting Parallax Occlusion Mapping (POM) with self-shadowing, emission, rim lighting, and approximate subsurface scattering.

PBR GGX Deferred — High-fidelity materials on the deferred pass
