Overview

Rendering realistic water in real-time requires a delicate balance between physical accuracy and performance. This Procedural Water Shader implements a comprehensive set of features to simulate the appearance of deep and shallow water entirely within a single shader pass.

Key highlights include vertex-level wave displacement using Gerstner waves, depth-aware color absorption (Beer-Lambert law), screen-space refraction, and dynamic foam generation based on object intersections and wave crests.

Gerstner Wave Synthesis

Unlike simple sine waves, Gerstner Waves provide a more realistic "peaked" shape by displacing vertices both vertically and horizontally. This prevents the "flat" look of basic water shaders and creates natural-looking wave crests.

HLSL 
float3 GerstnerWave(float4 wave, float3 p, inout float3 tangent, inout float3 bitangent) {
    float steepness = wave.z;
    float wavelength = wave.w;
    float k = 2 * PI / wavelength;
    float c = sqrt(9.8 / k);
    float2 d = normalize(wave.xy);
    float f = k * (dot(d, p.xz) - c * _Time.y);
    float a = steepness / k;

    tangent += float3(-d.x * d.x * (steepness * sin(f)), d.x * (steepness * cos(f)), -d.x * d.y * (steepness * sin(f)));
    bitangent += float3(-d.x * d.y * (steepness * sin(f)), d.y * (steepness * cos(f)), -d.y * d.y * (steepness * sin(f)));

    return float3(d.x * (a * cos(f)), a * sin(f), d.y * (a * cos(f)));
}

Depth & Absorption

To simulate the way light is absorbed as it travels through water, the shader samples the _CameraDepthTexture to calculate the distance between the water surface and the ground below.

Using the Beer-Lambert Law, we attenuate the light color based on this depth, resulting in natural transitions from clear, shallow shorelines to dark, opaque deep water.

Dynamic Foam Logic

Foam is generated in two primary ways:

  • Shoreline Foam: Created by comparing the depth of the water with the scene depth, adding white "bubbles" where the water is extremely shallow near geometric intersections.
  • Crest Foam: Uses the displacement height from the Gerstner waves to add foam at the tips of large waves, simulating breaking water.

Refraction & Fresnel

Screen-space refraction is achieved by offsetting the UV coordinates used to sample the _GrabTexture based on the water's surface normal. This creates the characteristic "underwater distortion" look.

The Fresnel Effect then blends between this refractive look and a reflective skybox color based on the viewing angle, making the water more reflective when viewed from grazing angles.

Performance & Optimization

Technique Optimization Benefit
Vertex Waves Limited wave count Low CPU/GPU overhead for animation.
Single Pass Consolidated logic Reduces draw calls and state changes.
Depth Masking Screen-space only Avoids complex volumetric calculations.
Normal Blend Dual-depth normal mapping Rich surface detail with minimal textures.
