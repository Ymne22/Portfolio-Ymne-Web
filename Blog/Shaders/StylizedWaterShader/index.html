Overview

The Stylized Water Shader is a comprehensive solution for creating vibrant, anime-style water in Unity's Built-in Render Pipeline. It combines procedural mesh generation with a rich set of features including toon shading, depth-based foam, screen-space reflections (SSR), and "anime sparkles" for that classic cinematic look.

Ocean Water Manager

The system is driven by the Scr_OceanWaterManager script, which handles the procedural generation of a subdivided plane. This allows for dynamic control over the water's scale and resolution directly from the inspector, while ensuring precise vertex animation for waves.

C# Snippet 
public void GenerateWaterMesh() {
    waterMesh = new Mesh { name = "Procedural_Ocean_Plane" };
    // ... calculate vertices based on subdivisions ...
    waterMesh.vertices = vertices;
    waterMesh.triangles = triangles;
    waterMesh.RecalculateNormals();
}

Toon Shading & Shadows

Unlike realistic water that relies heavily on complex Fresnel and PBR math, this shader uses Toon Stepping to create distinct color bands. It also supports receiving screen-space shadows, which are integrated into the water's surface without losing the "clear" aesthetic.

Screen Space Reflections

A custom SSR (Screen Space Reflection) pass provides real-time reflections of the world. By taking a GrabPass of the screen before the water renders, we can raymarch the reflection vector in screen space, allowing for dynamic reflections of characters and environments without expensive reflection probes.

Anime Sparkles

One of the defining features of this shader is the Anime Sparkle effect. It uses high-frequency procedural noise to generate sharp, shimmering highlights that react to the camera movement, mimicking the hand-drawn sparkles found in high-end anime water scenes.

Dynamic Caustics

The shader implements a dual-layer caustic system with chromatic aberration. By projecting two overlapping Voronoi patterns with different scales and speeds, it creates a complex, shifting light pattern that appears on both the water surface and objects below the depth threshold.
